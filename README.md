MyPhysicsLab
============

This it the source code for my Final Year Project.<br/>
Binary version available at: http://sourceforge.net/projects/myphysicslab

Physics Engine: Havok. <br/>
Rendering Engine: Ogre. <br/>
GUI Lib: MyGUI. <br/>

Features:<br/>
* 3D
* Real time
* Physics simulation
* Tracking camera
* Curve generating
